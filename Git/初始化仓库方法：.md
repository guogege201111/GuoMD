``` cpp

// 初始化
git init

// 添加
# 添加所有文件
git add .

# 或添加特定文件
git add 文件名

// 提交
git commit -m "初始提交"

// 关联远程仓库
git remote add origin 远程仓库URL
git branch -M main
git push -u origin main



## **确认分支状态**
# 查看所有分支
git branch -a

# 查看当前所在分支（前面带 * 号）
git status

## **切换到 main 分支**
git checkout main
## **拉取最新代码（如果与远程关联）**
git pull origin main
## **执行合并**
git merge master

##完成合并提交：
git commit -m "合并master到main分支"

## **推送到远程**
git push origin main
```
