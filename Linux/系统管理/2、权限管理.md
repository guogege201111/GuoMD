1、基础
```
1. 对象：u(owner)，g(group)，o(others)。
2. 权限：r(4, 读)，w(2, 写)，x(1, 执行)。
3. 数字权限是相加：rwx=7，rw-=6，r-x=5，r--=4。
   
   
   
755 = rwx r-x r-x

owner：可读可写可执行
group：可读可执行
others：可读可执行
  
  
  
  
  
644 = rw- r-- r--

owner：可读可写
group：只读
others：只读

```

2、查看组
```
whoami
groups
id

```

3、修改权限
```
chmod 644 notes.txt
chmod 755 run.sh
ls -l
chmod u+x notes.txt
chmod o-w notes.txt
ls -l notes.txt

```
