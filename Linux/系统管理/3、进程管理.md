1、核心概念
```
1. PID：进程号。
2. PPID：父进程号。
3. 前台进程：占用当前终端。后台进程：终端可继续输入命令。
4. 信号：TERM(15)优雅退出，KILL(9)强制终止。
```

2、前台运行及中断
```
//终端输入指令开始运行
./SkyOCT
// 中断
ctrl + C 
```

3、PS用法
```
// 查看SkyOCT的pid号
ps aux |grep SkyOCT

a：显示终端上的所有进程，包括其他用户的进程。
  
u：显示面向用户的格式（包含 %CPU、%MEM、VSZ、RSS 等）。
  
x：显示没有控制终端的进程。


// 按 CPU 使用率排序显示前几行
ps aux --sort=-%cpu |head
  
```

4、kill用法
```
// 优先使用优雅退出
kill -15 xxxxx

// 如果-15 不能完全退出
kill -9 xxxxx
```
