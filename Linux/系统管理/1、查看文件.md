1、tail
```
// 实时查看/var/log/syslog中有Error的记录
tail -f /var/log/syslog |grep Error
```

2、cat

```
// 将/var/log/syslog所有包含Mar的日志都输出到终端
cat /var/log/syslog |grep Mar
```

3、tee
```
// 将dmesg 输出的内核日志拷贝到/home/mc/log.txt
sudo dmesg -w -T |tee /home/mc/log.txt
```

4、less
```
less file.log             # 查看文件
less +100 file.log        # 从第100行附近开始
less +G file.log          # 直接跳到文件末尾
less +/ERROR file.log     # 打开后先定位到 ERROR
command | less            # 看命令输出（如 git log | less）

- q：退出
- Space / f：下一屏
- b：上一屏
- Enter / j：下一行
- k / y：上一行
- d：下翻半屏
- u：上翻半屏
- g：到文件开头
- G：到文件结尾
- 123g：跳到第 123 行
  
- /关键词：向下搜索
- ?关键词：向上搜索
- n：下一个匹配
- N：上一个匹配（与 n 反向）
- &关键词：只显示匹配行（过滤视图，再按 & 清空）

```

5、find
```
// 查找SkyOCT文件夹
find -type d -name "SkyOCT"
// 查找SkyOCT文件
find -type f -name "SkyOCT"
// 查找/home/mc/Project/文件夹内所有cpp
find /home/mc/Project/ -name "*.cpp"
```

6、whereis
```
// 查找可执行文件路径，只查系统常见目录，不是全盘扫描
whereis 程序名


-b 只看 binary
-m 只看 man
-s 只看 source
```

7、which
```
查看当前shell会执行哪个可执行文件，不加-a只显示搜到的第一个结果，加了-a显示全部结果
which python
# /usr/bin/python

which -a python
# /usr/bin/python
# /usr/local/bin/python

```
