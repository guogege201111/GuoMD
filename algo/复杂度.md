1、时间复杂度

O(1) 常数时间 ：无循环、无递归，或循环次数固定
```CPP
int a = 10;      // O(1)
int b = a + 5;   // O(1)
```
    
O(log n) 对数时间 ：循环变量每次翻倍或减半
```cpp
    for (int i = 1; i < n; i *= 2) {
    // O(1) 操作
}
```
O(n) 线性时间：单层循环，循环次数与 n 成正比
```cpp
    for (int i = 0; i < n; ++i) {
    // O(1) 操作
}
```
O(n log n) 线性对数时间：常见于快速排序、归并排序等分治算法。
    
 O(n²) 平方时间：双层嵌套循环
```cpp
    for (int i = 0; i < n; ++i) {
    for (int j = 0; j < n; ++j) {
        // O(1) 操作
    }
}
```
O(2ⁿ) 指数时间
    
O(n!) 阶乘时间

2、空间复杂度
O(1) 常数时间 ：只使用了有限的额外变量。
```cpp
int sum = 0;
for (int i = 0; i < n; ++i) {
    sum += i;
}
// 空间：sum、i、循环条件，常数级 O(1)
```
**O(n)**：使用了与 n 线性相关的额外空间，如数组、向量。
```cpp
int* arr = new int[n];   // 动态数组，占用 O(n)
// 使用 arr...
delete[] arr;
```
**O(n²)**：二维数组等。

**递归空间**：递归深度为 d，则栈空间为 O(d)。例如二分查找递归深度 O(log n)，空间 O(log n)；普通递归（如斐波那契）深度 O(n)，空间 O(n)。
